Usage Guide

Use API keys to authenticate proxy requests from Claude Code clients:

export ANTHROPIC_BASE_URL="http://<proxy-host>:9191" export ANTHROPIC_API_KEY="pck_..." claude

Keys created here are stored in MinIO and validated on each request. You can disable or delete keys at any time from this dashboard.
